Programme was created in BlueJ.

Purpose of the programme:
This application is to allow user add, remove and display details of DVD's in DVD's array.
Programme has no persistency.

How to:
1. After launching programme through BlueJ user can add, remove or display DVD's using menu system
2. Each menu option has it's number, user has to select option by typing a number on the keyboard and press enter to access
function
